# Tsakonian alphabet

The Tsakonian alphabet is a variant of the Greek alphabet with some extra sounds, written either with digrams or diacritics ([source](https://el.wikipedia.org/wiki/Τσακωνική_διάλεκτος#Γραφή)):

| Digram | IPA |
| --- | --- |
| σχ | [ʃ] |
| τσχ | [ʨ] |
| ρζ | [rʒ] |
| τθ | [tʰ] |
| κχ | [kʰ] |
| πφ | [pʰ] |
| τζ | [ʤ] |
| νν | [ɲ] |
| λλ | [ʎ] |

##### Other notes

* Some feminine nouns have a separate genitive singular, although this is rarely heard nowadays. It is formed using the suffixes -έ [e] ή -ή [i] (note that the accent moves to the final syllable). It is thought to derive from genitives in -ός in AG (third declension): e.g. νύξ, νυκτός, and its use has spread to first-declension nouns e.g. α προβάτα [a prováta], τα προβατέ [ta provaté]. The form in -ή is probably a development of that in -έ.
